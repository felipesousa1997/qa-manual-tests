# QA Manual Tests

Projeto de estudos em QA Manual utilizando o site Practice Test Automation.

## Objetivo

Validar a funcionalidade de login utilizando:

- testes positivos
- testes negativos
- edge cases
- validação de input

## Funcionalidades testadas

### Login Feature

Cenários executados:

- Login válido
- Usuário inválido
- Senha inválida
- Campos vazios
- Username com espaços
- Username muito grande
- Caracteres especiais
- Sensibilidade de maiúsculas/minúsculas

## Ferramentas utilizadas

- Excel
- GitHub
- QA Manual

## Status

✅ Todos os testes executados com sucesso.
